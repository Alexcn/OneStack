### 搭建开发环境

```shell
git clone https://github.com/itpubs/OneStack.git
cd OneStack
pyvenv venv
source venv/bin/activate
pip install -r requirements/dev.txt
```

### 代码风格建议

- 使用类视图代替函数视图
- 使用form来验证表单
